## Fixed-Bus Decision

Recommendation: pursue a follow-on implementation phase limited to small fixed caps.

Approved architectural direction:
- FLJ should expose plugin buses and let Ableton route tracks to and from those buses externally.
- Start with `Main` plus `Local Send 2` for local transmit inputs.
- Start with fixed extra output pairs exposed by FLJ for routed remote returns.
- Only consider broader bus counts after the first fixed-bus implementation is solid.

Explicit non-goal:
- Do not frame plugin-owned enumeration of Ableton tracks inside FLJ as the target implementation model.

Residual note:
- The current proof build still has only a minimal plugin-side route-selection experience for assigning a remote source to `Remote Out 1`.
- That is now an implementation-phase concern, not a research blocker.
