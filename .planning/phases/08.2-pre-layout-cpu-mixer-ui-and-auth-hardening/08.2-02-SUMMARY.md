## Decisions Made

- Fixed the room column at a conservative width in this phase so the layout stays predictable and testable while keeping scope out of Phase 9.
- Kept generic lifecycle status and added a dedicated inline auth label so password failures are explicit without weakening other status/error copy.
- Stored master output in wrapped plugin state and applied it after local monitor, remote returns, and metronome are summed so the footer control behaves like real output gain.
